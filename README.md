# MemMatch 🎮

一款色彩鲜明、简单直观的记忆配对游戏，适合各年龄段用户！

## 游戏特色

✨ **精美的卡片翻转动画** - 流畅的3D翻转效果
🎯 **多种难度级别** - 简单(3×4)、中等(4×4)、困难(4×5)
⏱️ **实时统计** - 计时器、尝试次数、配对进度
🎉 **视觉反馈** - 配对成功和失败的动画效果
⭐ **成绩评分系统** - 根据尝试次数给予星级评分
📱 **响应式设计** - 支持桌面和移动设备

## 如何开始

1. 在浏览器中打开 `index.html` 文件
2. 选择难度级别（简单、中等、困难）
3. 点击卡片进行翻转，找出所有匹配的对子
4. 尽可能用最少的尝试次数完成游戏！

## 游戏规则

- 每次可以翻开两张卡片
- 如果两张卡片图案相同，则配对成功，卡片保持翻开状态
- 如果两张卡片图案不同，则会自动翻回
- 找出所有配对即可完成游戏
- 游戏会记录你的用时和尝试次数

## 评分标准

- ⭐⭐⭐ 优秀：尝试次数接近完美
- ⭐⭐ 良好：尝试次数适中
- ⭐ 及格：完成游戏但尝试次数较多

## 技术栈

- HTML5
- CSS3（包含3D变换和动画）
- 原生JavaScript（无需外部依赖）

游戏包含以下界面：
